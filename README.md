# ✨ Yash Dagar | Premium Developer Portfolio



Welcome to the open-source repository for my **Premium Developer Portfolio**! This website is crafted to offer an immersive, first-person experience showcasing my skills, projects, and journey as a Software Engineer. 

---

## 🌟 Key Offerings

- 🎨 **Modern & Premium UI:** Incorporates sleek glassmorphism panels, deep neon gradients, and fluid CSS transitions.
- 📱 **Fully Responsive:** Offers a flawless, pixel-perfect experience across desktops, tablets, and smartphones.
- ✨ **Particle Backgrounds:** Uses `tsParticles` to generate an interactive, constellation-like background.
- 🚀 **Dynamic GitHub Projects:** Uses asynchronous JavaScript to fetch and display my latest projects directly from GitHub.
- 📧 **Web3Forms Integration:** A fully functional, honeypot-secured custom contact form ensuring messages are delivered right to my inbox.

---

## 🛠️ Technology Stack

| Technology | Purpose |
|------------|----------|
| **HTML5** | Semantic structure and accessibility |
| **CSS3** | Glassmorphism, animations, flexbox/grid layouts |
| **Vanilla JS** | DOM manipulation, GitHub API fetching, form handling |
| **tsParticles** | Interactive background animations |
| **FontAwesome** | High-quality vector icons |
| **Web3Forms** | Backend-less email service integration |

## 🚀 Running Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yashdagar56/yash-portfolio.git
   ```
2. **Navigate to the directory:**
   ```bash
   cd yash-portfolio
   ```
3. **Launch the project:**
   Simply double-click the `index.html` file to open it in your browser, or use a local server like VS Code's **Live Server** extension for a better development experience.
