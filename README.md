# Yash Dagar | Portfolio

This is a premium, highly-interactive portfolio website for Yash Dagar, an aspiring Software Engineer. The website features a beautiful first-person design, dynamic particle backgrounds, glassmorphism elements, and smooth animations.

## Features
- **Modern & Premium UI:** Glassmorphism, smooth animations, and neon gradients.
- **Responsive Design:** Fully responsive layout for mobile and desktop.
- **Particle Backgrounds:** Interactive tsParticles background.
- **Dynamic Projects:** Fetches projects dynamically from GitHub.
- **Functional Contact Form:** Integrated with Web3Forms for seamless client communication.

## Tech Stack
- HTML5
- CSS3
- JavaScript
- FontAwesome
- tsParticles

## Running Locally
To run the website locally, simply open the `index.html` file in any modern web browser or use a local development server like Live Server in VS Code.
